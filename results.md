# Per-cell sweep data

Backing data for [README.md](README.md). Generated by
[bench.py](bench.py) (sweeps) and
[evidence/decode_overhead.py](evidence/decode_overhead.py) (decode timing).

All numbers below are from `transformers 5.10.0.dev0` (upstream `main`
at commit `595721c`, 2026-06-03) on `torch 2.7.0+cu126`, A10G.

## Raw JSON schema

Per cell, each `logs/<run>.json` carries:
- `ttft_s_calls` — the 10 individual TTFT timings.
- `ttft_s_median`, `ttft_s_p99`, `ttft_s_mean`, `ttft_p99_over_p50`.
- `decode_total_s`, `decode_n_new`, `decode_tok_per_s`.
- `ss_artifacts_delta` — Inductor cache file count change during
  steady-state TTFT calls (should be 0).
- `ss_counters_delta` — `torch._dynamo.utils.counters` diff (empty
  when no new graphs/frames were captured).
- `warmup_*` — same fields for the warmup pass.

```sh
jq '.cells[] | {input_len, chunk_size, ttft_s_median, ss_artifacts_delta}' \
   logs/llama.json
```
